==================================
^= Bitwise Exclusive OR Assignment
==================================

Description
===========
Performs bitwise XOR and assigns value to the left operand.

Syntax
======
A ^= B

Remarks
=======
Equivalent to A = A ^ B.

Example
=======
>>> b = 0b1111
>>> b ^= 0b1010
>>> bin(b)
