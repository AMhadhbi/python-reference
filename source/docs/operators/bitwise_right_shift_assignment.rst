==================================
>>= Bitwise Right Shift Assignment
==================================

Description
===========
Performs bitwise right shift and assigns value to the left operand.

Syntax
======
A >>= B

Remarks
=======
Equivalent to A = A >> B.

Example
=======
>>> b = 0b1111
>>> b >>= 1
>>> bin(b)
