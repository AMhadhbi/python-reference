========================
^ (symmetric_difference)
========================

Description
===========
Returns a new set with elements in either the set or the other set but not both.

Syntax
======
**set ^ other**

*other*
    A set object or expression evaluating to a set.

Return Value
============
**set**

===============

Example
=======
>>> {1, 2} ^ {2, 3}
set([1, 3])
>>> {1, 2} ^ {1, 2}
set([])
