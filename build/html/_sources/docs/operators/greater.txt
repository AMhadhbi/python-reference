=========================
> Relational greater than
=========================

Description
===========
Returns True if x (left hand argument) is greater than y (right hand argument).

Syntax
======
A > B

Example
=======
>>> 1 > 0
True
>>> True > False
True
>>> 'b' > 'a'
True
>>> 'b' > 'B'
True
>>> 'b' > 100
True
